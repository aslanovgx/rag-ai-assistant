# 🤖 RAG AI Assistant

A full-stack AI assistant that combines a general chatbot with document-based Q&A using a Retrieval-Augmented Generation (RAG) pipeline.

Users can upload PDF/DOCX files and ask questions, receiving grounded answers with source context.

---

## 🚀 Live Demo

- 🌐 Frontend: https://rag-ai-assistant-c6f9.vercel.app/
- ⚙️ Backend:  https://rag-ai-assistant-cg9r.onrender.com
- 📚 API Docs: https://rag-ai-assistant-cg9r.onrender.com/docs

---

## ✨ Features

- General chatbot (LLM-based)  
- Document Q&A with RAG  
- PDF/DOCX upload  
- Semantic search with FAISS  
- Grounded answers with source chunks  

---

## 🧠 Tech Stack

**Backend:** FastAPI, LangChain, FAISS, HuggingFace, OpenRouter  
**Frontend:** Next.js, TypeScript, Tailwind CSS
